## <a name="overview">üìú Overview </a> 

The Apex Personal Portfolio Theme is a modern, clean, and creatively designed template tailored for creative individuals to showcase their portfolio. With a focus on highlighting your work and achievements, it provides an impressive design perfect for crafting an online presence.

## <a name="tech-stack"> </> Tech Stack </a> 

- **FontAwesome:** Provides a wide range of icons for enhancing visual elements.
- **Magnific Popup:** Enables the creation of responsive lightboxes and modals for showcasing content.
- **Owl Carousel:** Offers a responsive and touch-enabled carousel slider for displaying images or content.
- **Validate Form:** Facilitates form validation for ensuring data integrity and user experience.
- **Mixitup.js:** Allows for filtering and sorting of content with smooth animations.
- **SmartScroll.js:** Enhances scrolling behavior for improved user experience.
- **Bootstrap:** Provides a responsive grid system and pre-designed components for efficient web development.
- **jQuery:** Facilitates DOM manipulation, event handling, and AJAX requests.
- **wow.js:** Enables the creation of stunning animations and effects to captivate users.

## <a name="features">üëÄ Features </a> 

- Modern and clean design tailored for showcasing portfolios.
- Use of icons and visual elements from FontAwesome for enhanced presentation.
- Responsive lightboxes and modals powered by Magnific Popup for showcasing content.
- Interactive carousel sliders with Owl Carousel for displaying images or projects.
- Form validation using Validate Form for data integrity and user-friendly interactions.
- Filtering and sorting of content with smooth animations using Mixitup.js.
- Enhanced scrolling behavior for improved user experience with SmartScroll.js.
- Bootstrap framework for a responsive layout and pre-designed components.
- Stunning animations and effects created with wow.js to engage users.
